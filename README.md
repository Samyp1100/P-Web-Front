# Frontend - Gestión de Clínicas

Aplicación Angular para la gestión de clínicas.

## Tecnologías

- Angular 17
- TypeScript
- RxJS
- HttpClient

## Instalación

```bash
cd frontend
npm install
```

## Ejecutar

```bash
npm start
```

La aplicación estará disponible en `http://localhost:4200`

## Características

- Listado de clínicas
- Crear nuevas clínicas
- Eliminar clínicas
